
Instruções:

Para utilizar crie uma instancia da classe GithubApi e utilize o método de classe generate_file_from_dinda. Essa classe foi projetada para ter
métodos inseridos com outras conexões da api.

Observação:

O Giuthub pode retornar algumas informações pedidas no enunciado do exercício como nulas. O Hash "author" em alguns casos retorna nulo, com isso não
é possivel pegar o valor do "avatar_url" e "login".
